Here's some recent projects that I'm proud of

Project Details

Mark, Mangaging Director of Adm Plumbing Contractors, was spending three days a month collating all his plumber’s handwritten and typed timesheets into one document.

Mark approached me with the interest of automating this system and to find an easy way to view the time the company’s plumbers were spending on different jobs.

I worked with Mark to create a timesheet and plumber management web application from the ground up. The system enables all plumbers to log in and log their time spent on different jobs across the construction sites that the company is working on. Mark then had access to an admin panel controlling the entire system and enabling him to easily select a plumber or a plot to view all the associated jobs and logged times.

The software was built as a web application meaning that it can be accessed from any location on any device connected to the internet. The app was also made fully responsive meaning that other managers on site can access the app on their ipads.

I used PHP and a MySQL database to power the back-end of the application. In regards to the front end, this is powered by javascript, using the jQuery framework, HTML, using the bootstrap framework and CSS/LESS.

This application is constantly being improved and developed.
